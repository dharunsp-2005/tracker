# tracker
Educational criminal investigation tracking system

An educational project demonstrating a criminal investigation tracking system
with suspect prediction and immigration clearance simulation.

## Features
- Suspect probability prediction using Machine Learning
- Immigration clearance verification
- REST API using FastAPI
- Beginner-friendly project structure

## Tech Stack
- Python
- FastAPI
- Scikit-learn
- HTML
- GitHub

## Run the Project
```bash
pip install -r requirements.txt
cd backend
uvicorn main:app --reload
